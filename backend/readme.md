## Usage Guide

1. Set up and Activate Python Virtual Environment

Unix
```
$ python3 -m venv venv
$ source venv/bin/activate
(venv) $ _
```
or

Windows
```
$ python -m venv venv
$ venv\Scripts\activate
(venv) $ _
```

2. Install Dependencies
```
pip install -r requirements.txt
```

3. Run!
```
flask run
```
